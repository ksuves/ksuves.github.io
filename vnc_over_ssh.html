Доступ к рабочему столу через VNC поверх SSH

Нужен VNC-сервер и VNC-клиент. В качестве VNC-сервера x11vnc показал себя лучше, чем tigervnc с Openbox. Цель — получить доступ к рабочему столу server с машины client. 

client$ ssh -L 5900:localhost:5900 -l user server
user@server$ startx
# На машине server от пользователя user должны быть подняты иксы.
# Если они уже запущены другим пользователем, можно использовать:
startx -- :<№>
user@server$ x11vnc -localhost -display :0 -nopw
# Соответственно, -display :<№>
client$ vncviewer localhost
